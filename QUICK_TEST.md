# ⚡ راهنمای تست فوری

## یک دقیقه‌ای برای تست کردن ✅

### ⏱️ مرحله 1: صفحه (30 ثانیه)
```
1. وارد سایت شوید
2. یک محصول را انتخاب کنید
```

### ⏱️ مرحله 2: Console باز کنید (10 ثانیه)
```
Windows/Linux: F12 → Console tab
Mac: Cmd+Option+I → Console tab
```

### ⏱️ مرحله 3: Refresh (10 ثانیه)
```
Ctrl+R یا Cmd+R
```

### ⏱️ مرحله 4: لاگ‌ها بررسی کنید (5 ثانیه)
```
دنبال کنید:
✓ Event listener attached to main add-to-cart button
✓ Event listener attached to mobile add-to-cart button
```

### ⏱️ مرحله 5: کلیک و تایید (5 ثانیه)
```
دکمه "افزودن به سبد خرید" را کلیک کنید
✓ Modal ظاهر می‌شود
✓ سبد خرید به روز می‌شود
```

---

## ✅ موفقیت یعنی:

- ✓ Modal ظاهر شود
- ✓ محصول در سبد خرید باشد
- ✓ لاگ‌های console بدون ❌ باشند

---

## ❌ مشکل یعنی:

- لاگ "Event listener attached" ظاهر نشود
- هنگام کلیک، هیچ اتفاقی نیفتد
- ❌ علامت‌های خطا در console

### اگر مشکل داشت:

1. بررسی کنید آیا لاگ‌های initialize ظاهر شده‌اند
2. بررسی کنید آیا لاگ‌های "button not found" ظاهر شده‌اند
3. فایل `DEBUG.md` را بخوانید
